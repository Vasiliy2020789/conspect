Переменная - это имя с которым связано значение. Предназначена для хранения данных для последующего использования их в коде. Значение переменой можно изменять в коде.

Имя переменной может содержать только буквы, лучше латинского алфавита, цифры и символы $ _ При этом не может начинаться с цифры. В качестве имен переменных нельзя использовать зарезервированые имена, они используются самим языком JS.

Читаемость кода.

Для лучшей читаемости кода принято

  • Давать понятные имена переменной по которым можно иметь представление о её содержимом
  • Применять стиль lowerCamelCase он же горбатый регистр или Верблюжий регистр если переменная состоит из нескольких слов
  • Лучше использовать латинские буквы

Объявлять несколько переменных можно через запятую но лучше каждую с новой строки.

Значение переменной можно менять по ходу кода.

Одной переменной можно присвоить значение другой переменной

Константы

Для объявлениея константы используется КС const. Значение константы нельзя изменять в рамках одного блока, в остальном тоже что и переменная. Но в массивах и объектах можно изменять значение констант см пример 12

Отличия let и var

Ключевое отличие в том что переменную var можно использовать еще до её объявления в коде независимо строгий режим или нет.

Второе отличие в облости видимости. Перен=менная var объявленная внутри блока IF или в цикле будет видна за его пределами. Но если переменная объявлена внутри функции то за пределами функции её существовать не будет, так-же как и переменной let.
