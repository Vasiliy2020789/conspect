Типы данных в JavaScript

Тип данных - это набор характеристик определяющий тип поведения переменоой

JavaScript является динамически типизированым языком - это значит тип данных меняется динамически в момент присвоения или изменения значения переменной а не в момент ообъявления переменной.

Всего 9 типов данных:

  • Undefined (неопределенный - переменнаяобъявлена но не определена) пример 1
  • Null (Пустое место, пееременная определена но её значение - ничего) пример 2
  • Boolean (Логический тип, true или false) пример 3
  • Number (целые значения а так же числа с плавающей запятой, Infinity пример 4.1, NaN пример 4.2) пример 4
  • Bigint (число большее чем 9007199254740991 и) пример 5
  • String (строка - текстовое значение любой длинны) значение задается в кавычках(могут быть 3х типов пример 6 "" пример 6.1'' пример 6.2 ``
    • "" и '' равнозначны
    • `` - можно встраивать переменные пример 6.3
      let userAgeInfo = `Возраст: ${userAge} лет`;
  • Symbol (Уникальный идентификатор обекта) пример 7
  • Object (содержит сложную структуру данных в виде свойств. Свойство - это пара ключ и значение) пример 8
  • Function (похож на Object) пример 9
