Пример 35 .classList - специальный объект с методами для добавления/удаления одного класса

Методы объекта .classList :
  • .add - добавляет класс
  • .remove - удаляет класс
  • .toggle - добавит класс если его нет а если есть то удалит
  • .contains - проверяет наличие класса, возвращает true/false

Свойства и атрибуты DOM элементов

У разных DOM элементов могут быть разные свойства. например у тега a - свойства связанные со ссылкой; у тега input - cd-df связанные с полем ввода и т.д.

В HTML у тегов могут быть атрибуты. Когда браузер парсит HTML, чтобы создать DOM-объекты для тегов, он распознает стандартные атрибуты и создает DOM свойства для них

Каждый DOM узел принадлежит соответствующему встроенному классу

Подробнее про классы DOM-узлов: https://learn.javascript.ru/basic-dom-node-properties#klassy-dom-uzlov
