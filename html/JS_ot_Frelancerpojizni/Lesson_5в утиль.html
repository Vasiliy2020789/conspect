Операторы в JavaScript

Операторы в JavaScript делятся на 3 группы

Операнд или аргумент - это то к чему применяются операторы

  • Базовые операторы (+ - * / **) Унарные пример 1 - применяются к одному и бинарные пример 2- применяются к двум операндам
    • + пример 3
    • - пример 4
    • * пример 5
    • / пример 6
    • % пример 7
    • ** пример 8
  • Операторы сравнения
    • Больше: a > b
    • Меньше: a < b
    • Больше или равно: a >= b
    • Меньше или равно: a <= b
    • Равно: a == b
    • Не равно: a != b
    • Строго равно: a === b
    • Строго не равно: a !== b
  • Логические операторы
    • ИЛИ ||

Логические операторы.

Оператор ИЛИ. ||

Пример 25.1. ИЛИ ||
1. Вычисляет операнды слева направо.
2. Каждый оеранд конвертируется в Bollean, если результат true, останавливается и возвращает исходное значение этого операнда.
3. Если все false то возвращает значение последнего из них.

console.log('//Пример 25.1 ИЛИ ||. Булевые значения');
console.log(true || true);
console.log(false || true);
console.log(true || false);
console.log(false || false);

Пример 25.2 ИЛИ ||. Разные типы данных.'

console.log(1 || 0);
console.log(true || 'Фрилансер');
console.log(null || 58);
console.log(null || 'Фрилансер' || 0);
console.log(undefined || '' || null || 0);

Оператор И. &&

1. Вычисляет операторы слева направо.

2. Каждый операнд преобразуется в логическое значение. Если результат false, останавливается и возвращает исходное значение этого операнда.

3. Если все операнды были истинными возвращает значение последнего.

Пример 26.1 И &&. Булевые значения

console.log(true && true);
console.log(false && true);
console.log(true && false);
console.log(false && false);

Пример 26.3 И. Разные типы данных.'

console.log(' Фрилансер' && 0 && 2 && 3);//0 - false результат 0
console.log(1 && 2 && null && 3);//null - false результат null
console.log(15 && 42);//0 - false результат 42

Пример 26.4 Приоритет && больше чем ||

console.log(1 && 0 || 1 && 2);

Оператор НЕ !

1. Сначала приводит аргумент к логическому типу true или false.

2. Затем возвращает противоположное значение.
