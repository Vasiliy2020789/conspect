Пояснение параметра

Задает границу блочного тега. Работает только для блочных тегов и картинок. К строчным тегам не применяется.


Возможные значения:
Толщину границы можно задавать в пикселях, а цвет в любой системе #, rgb или rgba.
Типы границы могут быть такие: solid | dotted | dashed

Значение: 1px solid #000;

Абревиатура EMMET

bd + Tab = border: 1px solid #000;

Прозрачность элемента

opacity:

Задает прозрачность элементу

Значение задается в долях 0 - полная прозрачность, 1 - нет прозрачности. В отличии от display: none; блок не убирается из верстки а просто становится прозрачным. С прозрачными элементами также можно взаимодействовать, ссылки и кнопки, также продолжают работать.

Абревиатура EMMET

box + ищем в выпадающем списке и выбираем box-shadow: и вводим параметры

.example-block-11 {
border: 1px solid #000;
opacity: 0; /*Элемент не видим, но находится на странице*/
}
.example-block-12 {
border: 1px solid #000;
opacity: 0.5;/*Задает прозрачность 50%*/
}

Значение по умолчанию visible. При скрытии элемента данным свойством место занимаемое им резервируется, взаимодействовать с элементом нельзя.

}

visibility: hidden; /*Элемент скрыт*/
