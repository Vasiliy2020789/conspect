HTML формы.

HTML формы добавляют сайту интерактивности. Например формы поиска, регистрации, авторизации. С помощью форм данные введеные пользователем передаются на сервер.

HTML теги необходимые для создания форм.

<form></form> - блочный тег, оболочка формы.Все поля, переключатели, чекбоксы пишутся внутри этого тега.
Атрибут action="" определяет куда передавать данные.
Атрибут method="" может принимать значение "get" - по умолчанию, передача данных идет прямо в адресной строке подходит для передачи малого количества данных и метод "post" - передает данные скрытно используя запрос браузера.

Основные атрибуты тега <></form>

Если мы хотим чтобы наша форма умела отправлять прикрепленные файлы то нужно использовать:
<form action="#" method="post" enctype="multipart/form-data"></form>

Значение атрибутов: type="text" - тип данных. Возможные значения text (по умолчанию) ввод текста, tel - переключит клавиатуру мобильного устройства пользователя на цифры, type="password" - скрывает вводимый текст. name="username" value="" Общие атрибуты форм. Для тегов <input/> и <textarea/> можно задать атрибут placeholder="" placeholder="Подпись кода" задает надпись внутри текстового поля.
