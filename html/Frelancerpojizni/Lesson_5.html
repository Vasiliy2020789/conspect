Каскадные таблицы стилей - CSS. Обнуление CSS стилей.

CSS описывает как наша страница будет выглядеть. Задает цвета, шрифт, размеры и положение элементов, их поведение при адаптации. Используется для создания анимации на странице.

В этом уроке мы создадим css файл, подключим его к странице, рассмотрим синтаксис, классы, к концу урока получим т.н. обнуляющий css файл, с которым будем работать дальше.

Подключение css файла.

Создаем новый файл, пользуясь сочетанием клавиш Ctrl + N и тутже сохраняем его нажимая Ctrl + S в папку ..//css/style.css

Далее нужно подключить файл стилей к странице. Для этого пишем в теге <head> <link rel="stylesheet" type="text/css" href="../../css/style.css">

Синтаксис css очень простой, нужно обратиться к тегу или группе тегов в нашем html документе и описать в них нужные стили.

Если обратиться к названию тега, например div то ко всем тегам div применятся описаные стили.

Если обратиться к классу то ко всем элементам указанного класса применятся описаные стили.

Таке можно обращаться к тегу и классу одновременно

В CSS файле это выглядет вот так:

Давайте выделим текст в тегах span красным и сделаем его крупнее:

span{
font-size: 20px;
color: red;
}

Задает синий цвет классу .hot_key

Задает оранжевый цвет всем параграфам с классом css_code. У такого вида селектора приоритет выше чем если бы мы обратились только к классу или только к тегу. Не зависит от места расположения в каскаде.

p.css_code{

Вложенность.

Задача: Есть 2 параграфа, первый с классом .bigred второй без класса. Сделать зеленым текстом span-ы внутри класса .bigred
Решение:

.bigred span{
font-size: 20px;
color: #00ff00;
}

Наследование параметров.

Параметры могут наследоваться тегами находящимися внутри стилизированого элемента, но не все.

Например

У нас есть div внутри которого есть 2 параграфа и есть 2-й div в котором есть параграф. Зададим ему несколько стилей:

.block_nas{
color: blue; /*наследуется*/
font-size: 18; /*наследуется*/
border: 3px solid #000; /*НЕнаследуется*/
}

Первый параграф внутри первого тега div. Lorem ipsum dolor sit amet consectetur adipisicing, elit. Minus architecto, amet deserunt odio omnis, repellat explicabo illum animi quod quidem sed iure dolore culpa sit placeat minima. Veniam necessitatibus, incidunt.

Второй параграф внутри первого тега div. Lorem ipsum, dolor sit amet consectetur adipisicing elit. Ipsam.

TEXT. Параграф внутри второго тега div.

Как видим цвет и размер текста наследовались вложеными тегами а вот рамка присвоилась только к родительскому элементу то есть к первому тегу div. Более подробно наследование того или иного параметра мы изучим в ходе практики. И к тому же вместо css мы в далнейшем будем использовать препроцессор SAS в синтаксисе scss, это значительно упростит нам работу.

Обнуляющий css файл

У разных тегов по умолчанию присутствуют разные стили, отступы размеры шрифта и т.п. и каждый браузер отображает стили по своему. Это не соответствует требованиям по верстке и не будет совпадать с макетом. Обнуляющий css файл нужен для того чтобы убрать все эти стили.
