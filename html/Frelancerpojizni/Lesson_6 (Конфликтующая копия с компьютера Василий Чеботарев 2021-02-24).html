Домой

CSS Стили шрифта и текста. CSS font-size. CSS color, CSS text-shadow.

Стилизация текста это одно из основных предназначений css.

В этом уроке мы изучим и научимся применять стили шрифта и текста. А в конце урока выполним практическое задание из макета PSD.

Терминология.

span{ /* селектор типа*/
font-size: 20px; /* (свойство: значение) вся запись называется параметр*/
color: red; /* (свойство: значение) вся запись называется параметр*/
}

.hot_key { /* селектор класса*/
color: #0000ff
}

p.css_code{ /* селектор типа и класса */
color: #FF8A00;
}
